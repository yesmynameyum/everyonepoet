# ✒️ 누구나 시인 (Everyone is a Poet)

> **"매일 주어지는 감정의 언어로 오늘의 내면을 짓고, 1년 뒤 과거의 나에게 답장을 보냅니다."**

**누구나 시인**은 하루에 단 한 번 주어지는 감정의 언어를 통해 내면을 기록하고, '당일 열람'이라는 제약을 통해 순간의 감정에 온전히 몰입하게 하며, 1년 뒤 과거의 나에게 답시를 쓰며 정서적인 성찰을 돕는 감성적 웹 애플리케이션입니다.

---

## 🌌 5W1H 제품 개요

* **Why (목적)**: 
  바쁜 현대 사회에서 흩어지는 감정을 정제된 '시(Poetry)'의 형태로 기록하게 합니다. '당일 열람 제한'이라는 시간적 제약 장치를 통해 기록하는 순간에 온전히 몰입하도록 유도하고, 1년 뒤 오늘의 글에 답시를 작성하게 함으로써 정서적 깊이와 내면의 성장을 확인합니다.
* **Who (타겟)**: 
  시와 문학을 사랑하는 사람, 매일 꾸준히 감정 일기를 쓰고 싶지만 형식이 막막한 사람, 과거의 나와 정서적인 대화를 나누고 싶은 사람.
* **What (핵심 산출물)**:
  - 감성적인 타이포그래피 기반의 서비스 랜딩 페이지
  - 365일 매일 다르게 주어지는 감정 테마 기반의 작시 에디터
  - 타이핑 중 유실을 방지하는 3초 실시간 자동 저장 로직
  - 밤 12시가 지나면 글의 본문이 잠기는 당일 열람 제한 기능
  - 2년 차(가입 후 366일 이후) 진입 시 1년 전 시를 해제하고 답시를 쓸 수 있는 듀얼 화면
* **When (사용 상황)**: 
  하루를 마무리하며 내 안의 감정을 차분히 정돈하고 싶을 때 (하루 1회 작성 및 당일만 열람 가능).
* **Where (환경)**: 
  오롯이 글쓰기에 집중할 수 있는 깔끔한 UI의 데스크톱 및 모바일 웹 브라우저.
* **How (구현 방식)**: 
  Vanilla HTML, CSS, JavaScript 및 LocalStorage 가상 데이터베이스 구조. 외부 서버가 없어도 브라우저 캐시 환경에서 완전히 자립적으로 동작.

## 🎨 주요 기능 상세

### 1. 시적인 다크 네이비 테마 & 글래스모피즘
- 우주의 깊은 밤을 테마로 한 그라데이션과 반짝이는 별빛 백그라운드 애니메이션을 내장하였습니다.
- 반투명 글래스(`backdrop-filter: blur(16px)`)와 넉넉한 글자 간격, `Noto Serif KR` 명조체 폰트로 고요하고 정갈한 시인의 다이어리를 시각화했습니다.

### 2. 실시간 임시 자동 저장 (Auto-Save)
- 에디터 작성 중에 3초간 입력이 멈추면 자동으로 브라우저에 백업을 저장합니다.
- 예기치 않게 새로고침을 하거나 창을 닫아도, 오늘 날짜에 작성 중이던 글을 그대로 복구합니다. 발행 버튼을 누르기 전까지는 계속 안전하게 복사본을 보관합니다.

### 3. 밤 12시 봉인 메커니즘 (Locking Rule)
- 오늘 쓴 시는 당일 밤 11시 59분 59초까지만 조회 및 수정이 가능합니다.
- 자정이 되는 즉시(실시간 타이머 동작) 과거의 글로 판정되어 본문이 숨겨지고 아카이브 카드에 자물쇠가 걸립니다.
- 자물쇠를 누르면 1년 뒤 오늘(작성일+365일)까지 남은 디데이가 은은하게 표기됩니다.

### 4. 2년 차 답시 작성 (Dual View)
- 가입 후 1년이 경과하는 순간(가입일 기준 366일 차), 정확히 365일 전에 썼던 과거 글의 자물쇠가 해제됩니다.
- 화면은 1년 전의 글을 보여주는 '과거 패널'과 오늘 이에 응답을 적는 '답시 패널'로 분할되어, 1년 시간의 공백을 두고 스스로와 감정적 대화를 나눌 수 있습니다.

## 🛠️ 개발자 시뮬레이터 내장 (Demo Simulation)
시간 제약이 걸린 플랫폼 특성(자정 잠금, 1년 뒤 해제)상, 동작 확인을 돕기 위한 **데모 시뮬레이터 위젯**이 우측 하단에 기본 탑재되어 있습니다.
- **가상 시스템 시각**: 날짜를 내일이나 미래로 변경하여 오늘 쓴 시가 자물쇠로 잠기는지 즉시 검증할 수 있습니다.
- **가상 가입 날짜**: 가입일을 '365일 전'으로 설정하여, 즉시 '2년 차 시인' 듀얼 에디터 탭으로 넘어가는 상황을 연출할 수 있습니다.

## 📂 실행 및 설치 방법

본 프로젝트는 별도의 서버나 데이터베이스 설정, 빌드 패키지 없이 독립적으로 실행 가능합니다.

1. 본 저장소를 클론하거나 다운로드합니다.
   ```bash
   git clone https://github.com/YOUR_GITHUB_ID/repository-name.git
   ```
2. 다운로드한 폴더의 `index.html` 파일을 크롬, 엣지, 사파리 등의 브라우저로 직접 엽니다.
3. 로컬 호스트 서버를 띄워 확인하고 싶으시다면 아래의 방법 중 하나를 선택해 활용하세요.
   - **Node.js**: `npx http-server` 실행 후 `localhost:8080` 접속
   - **Python**: `python -m http.server 8000` 실행 후 `localhost:8000` 접속
